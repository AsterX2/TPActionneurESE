Description détaillée

Gestion du contrôle de la vitesse moteur par PWM.

Auteur
MundusEst
Date
13 Novembre 2024

Ce fichier contient les fonctions de contrôle du moteur via PWM, permettant de démarrer, arrêter et réguler la vitesse du moteur. Il gère également la mesure du courant via l'ADC.

Documentation des fonctions

◆ start_PWM()

void start_PWM ( TIM_HandleTypeDef htim,
uint32_t channel )

Démarre la génération PWM.

Cette fonction configure et démarre la génération PWM sur les deux canaux en mode complémentaire. Elle initialise également les rapports cycliques à 50% pour un démarrage équilibré.

Paramètres
htimHandle du timer utilisé pour la génération PWM
channelNuméro du canal PWM à configurer

◆ stop_PWM()

void stop_PWM ( TIM_HandleTypeDef htim,
uint32_t channel )

Arrête la génération PWM.

Cette fonction arrête la génération PWM sur le canal spécifié ainsi que sur sa sortie complémentaire. Elle est utilisée pour arrêter le moteur en toute sécurité.

Paramètres
htimHandle du timer utilisé
channelNuméro du canal PWM à arrêter
